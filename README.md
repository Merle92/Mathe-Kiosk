# Mathe-Kiosk

Mathe-Kiosk ist eine interaktive Webanwendung zur Förderung mathematischer Kompetenzen im Grundschulbereich.  
Die Anwendung ermöglicht es, Einkaufssituationen spielerisch nachzustellen und Rechenstrategien im Umgang mit Geld zu üben.

## Zielsetzung

Mathe-Kiosk unterstützt Schülerinnen und Schüler beim:

- Addieren und Subtrahieren von Geldbeträgen  
- Verstehen von Münz- und Scheinwerten  
- Berechnen von Wechselgeld  
- Anwenden mathematischer Strategien im Sachkontext  

Die Anwendung ist für den Einsatz im Unterricht sowie im Studium der Grundschulpädagogik konzipiert.

---

## Nutzung

Die Nutzung von **Mathe-Kiosk** ist im schulischen und hochschulischen Kontext ausdrücklich erwünscht.
